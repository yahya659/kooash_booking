منصة حجز كوش الأعراس

منصة لعرض الكوشات وحجزها، مع إمكانية تحديد تاريخ العرس وإرسال رسائل للمكتب، إضافةً إلى لوحة تحكم لإدارة الموقع بسهولة.

 الميزات:

عرض الكوشات بالصور والتفاصيل

حجز الكوشة وتحديد تاريخ العرس

نظام رسائل الزوار للمكتب

لوحة تحكم Dashboard لإدارة المحتوى

تصميم متجاوب لجميع الأجهزة

 التقنيات:

Laravel 10

PHP 8+

MySQL

Bootstrap 5

Blade Templating

⚙️ التثبيت
git clone https://github.com/your-username/kosh-wedding-platform.git
cd kosh-wedding-platform
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
